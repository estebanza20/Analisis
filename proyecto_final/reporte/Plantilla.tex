\section*{Resumen}
Este trabajo trata sobre el modelado de un robot de dos ruedas. Este es un sistema robótico que funciona como un péndulo inverso. Se utilizan las leyes de dinámica ya existentes para obtener un modelo matemático que describa al sistema, y posteriormente se simula en MATLAB, para verificar que el modelo describa correctamente al sistema.

\section[Introducción]{Introducción}

El sistema qeu se estudia en el presente trabajo es un robot de dos ruedas, esto significa que toda la estructura robótica se sostiene únicamente sobre dos ruedas. El robot únicamente tiene dos puntos de contacto con el suelo (uede ser una línea, depende del tipo de ruedas), lo que hace que se comporte como un péndulo inverso. Es naturalmente inestable. En la figura %ref a la figura
se muestra un robot ejemplo. 

Lo que intereza del sistema es mantenerlo erguido, por tanto el modelo se hará tomando en cuenta este fin. Además el análisis del sistma se toman en cuenta las siguientes suposiciones:


\begin{itemize}[labelindent=1cm,labelwidth=2.25cm,align=left]
%ponerlo en orden alfabetico
	\item El robot se encuentra sobre una superficie plana y estática.
        \item No hay deslizamiento de las ruedas. Siempre hay fricción estática.
        \item Los motores puede aplicar torque infinito.
        \item Ambas ruedas se mueven con la misma velocidad.
        \item La única fuerza externa es la fuerza de la gravedad.
        \item Las medidas de los sensores se consideran ideales, sin ruido, o ya filtradas.
        \item 

\end{itemize}
